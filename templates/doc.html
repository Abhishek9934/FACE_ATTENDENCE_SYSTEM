Documentation

Introduction

  • Item Name : Face Attendance System
  • Item Version : v 1.0

This documentation is to help you regarding each step of usage. Please go through the documentation carefully to understand how this face attendance system works to make full use of this service according to your needs. The app provides separate portals for students and faculty.



Faculty Instructions

  1. Faculty in order to use this app first needs to register on the application.


  2. After creating the account faculty can login to his account.


  3. In the teacher panel faculty can create his new class / delete his existing class / register a new class / mark daily attendance.




  4. After creating a new class the teacher can register students in it by uploading single class photo.




  5. Teacher can view the class details by clicking on a particular class.




  6. Click the mark attendance button and upload a class photograph.
  7. Select the class for which attendance to be marked and then click on the submit button and the attendance will be marked.




  8. This opens a editable form with attendance marked for all the students present in the photograph.Incase there are any miss-detections then the teacher can edit the attendance form.




  9. The user can edit/delete or add a single student.





Student Instructions

  1. Students in order to use this app first needs to register on the application.


  2. After creating the account student can login to his account.




  3. In the student panel he/she can view all his enrolled classes.




  4. He can see the details on his class home page such as attendance details.




  5. The student can register anytime even after a long time to view all his attendance details.
